📋 App Guidelines

Government of India–aligned guidelines compiled from advisories, policies, and safety frameworks for women, digital apps, and public safety systems.

1. User Safety & Emergency Response
  • Must provide quick emergency access (SOS/panic button)
  • Should enable real-time location sharing with authorities/contacts
  • Must integrate emergency helplines (100, 112, etc.)
  • Systems should ensure immediate response by police or services
2. Data Privacy & Security
  • Must protect user identity and personal data
  • Should follow consent-based data collection
  • Avoid unnecessary access to sensitive data (calls, storage, etc.)
  • Ensure no misuse or surveillance concerns
3. Accessibility & Ease of Use
  • App must be simple, fast, and usable in panic situations
  • Should work with minimal steps and low user effort
  • Must support multiple languages (including regional languages)
  • Should be inclusive for women, children, elderly
4. Reliability & Availability
  • Must function in low network or offline mode (SMS fallback)
  • Should provide accurate GPS tracking
  • Must ensure high uptime and quick response systems
5. Integration with Authorities
  • Should be linked with police control rooms / emergency response systems
  • Must support tracking, alert monitoring, and quick dispatch
  • Align with women help desks & emergency response initiatives
6. Trust & Safety Assurance
  • Ensure verified systems and secure backend
  • Avoid false alerts and misuse
  • Build user trust through transparency and reliability
7. Special Women Safety Measures
  • Features like female support options / safer services encouraged
  • Focus on preventing crimes and ensuring quick reporting
  • Promote safe environment policies and awareness

🔌 Required API Integrations
1. Browser Geolocation API
Built-in browser API · No token needed
navigator.geolocation.getCurrentPosition()
2. Google Maps / Directions API
Route calculation · Distance · ETA
🔑 Token: Google Cloud Console → Maps → Directions API Key
https://maps.googleapis.com/maps/api/directions/json?key=YOUR_KEY
3. Google Places API (Nearby Search)
Find nearest police station · Name · Address · Phone
🔑 Token: Google Cloud Console → Places API Key
https://maps.googleapis.com/maps/api/place/nearbysearch/json?type=police&key=YOUR_KEY
4. OpenStreetMap / Overpass API
Free alternative for police station lookup · No token needed
https://overpass-api.de/api/interpreter?data=[out:json];node[amenity=police](around:5000,LAT,LNG);out;
5. Nominatim Reverse Geocoding
Convert GPS coords → district/state name · Free, no token
https://nominatim.openstreetmap.org/reverse?lat=LAT&lon=LNG&format=json
6. DIAL 112 / Emergency API (India)
Ministry of Home Affairs — official emergency integration
🔑 Token: Apply at dial112.gov.in — Government partnership required
POST https://api.dial112.gov.in/v1/alert { lat, lng, userId }
7. SMS Gateway (Twilio / MSG91)
Send SOS SMS alerts to emergency contacts offline
🔑 Token: twilio.com or msg91.com → API Key + Account SID
POST https://api.twilio.com/2010-04-01/Accounts/SID/Messages
Voice Commands: Off
